# README

## Installation phase

* install ruby gems

`bundle install`

* install js packages

`yarn install`

* setup the DB

`rake db:setup`

* add fake transactions by running db seed

`rake db:seed`

* run rails server

`bin/rails server`

* run webpacker on different terminal

`bin/webpack-dev-server`

* visit localhost:3000

* sign up a user account
